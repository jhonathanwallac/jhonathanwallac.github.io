Sobre mim 👨‍🚀

Olá,
eu sou o Jhonathan :)

Estudante de Sistemas de Informação na Universidade Federal dos Vales do Jequitinhonha e Mucuri (UFVJM) e colaborador da empresa ATS Informática na função de desenvolvedor estagiário, assegurando o funcionamento ideal do sistema web fazendo o uso principalmente das tecnologias Java e PostgresSQL.

Com objetivo de adquirir habilidades tanto no back-end quanto no front-end, ao mesmo tempo, dedica o tempo no estudo e desenvolvimento de projetos pessoais nas áreas de UI/UX Design e Desenvolvimento Front-end, usando ferramentas como Figma, HTML, CSS e JavaScript para criar interfaces atraentes e interativas.

Além do conhecimento em desenvolvimento de software, também possui familiaridade com metodologias ágeis, especialmente o Scrum. Tendo em vista que a adoção de metodologias ágeis é fundamental para garantir a colaboração eficaz entre equipes e a entrega de produtos de alta qualidade dentro dos prazos estabelecidos.

Projetos favoritos ⭐

Snake Game

Este projeto é uma tentativa de reprodução do clássico jogo Snake, presente em diversos celulares da marca Nokia por volta do ano 2000. Este é um jogo em que o usuário controla uma cobra que cresce à medida que come alimentos, evitando obstáculos e tentando não colidir com a própria cauda. O jogo inclui recursos como pontuação, níveis de dificuldade e controles intuitivos.

Login Page

Este projeto é uma tela de login responsiva e elegante que permite ao usuário acessar um sistema ou aplicativo com segurança. Weather app

Este é um projeto simples e intuitivo que permite aos usuários consultar informações sobre o clima em várias cidades ao redor do mundo. Com um design limpo e amigável, o site oferece uma experiência agradável para os usuários. Ao acessá-lo, os usuários são recebidos com uma barra de pesquisa onde podem inserir o nome da cidade desejada, em seguida, o site exibe imediatamente os dados meteorológicos atualizados da localização pesquisada.
